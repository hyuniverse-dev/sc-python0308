<설명문>

- 파일 시스템을 활용한 회원 관리 프로그램

 Pyhton + JSON 파일을 활용해서 사용자 정보를 저장하고 관리하는
 간단한 회원 관리 프로그램이다. JSON 파일을 데이터베이스 대신하여 저장소로 사용해
 사용자 정보를 영속화하여 사용한다.

- 주요 기능

 1. 회원가입: 아이디, 비밀번호, 이름, 이메일을 입력받고
             유효성 검사(이메일, 비밀번호)를 통해 데이터가 올바른지 확인한다.
             아이디가 중복되지 않으면 사용자 정보를 JSON 파일에 저장한다.
 2. 로그인: 아이디와 비밀번호를 입력받아 JSON 파일에서 일치하는 사용자 정보를
           조회한다.
 3. 회원탈퇴: 아이디와 비밀번호 확인하여, 일치하는 사용자를 JSON 파일에서 삭제한다.


- 학습 목표

 1. 파이썬의 객체지향 프로그램에 대한 개념을 이해한다.
 2. 파일 시스템 활용을 이해한다.
 3. try-except 문 활용을 이해한다.
